### 우리 프로젝트는❓
> mbti를 기반으로 한 유리병 편지 전송 서비스입니다.

## 주요 기능
📑 member 기능
- 로그인/회원가입
- mbti 테스트
- mbti 관련 테스트 참여/ 결과 통계 조회
- mbti 별 관련 문제 결과 조회
- mbti 랜덤 편지 발송/답변
- 특정 편지 공유
  
📑 admin 기능
- mbti 관련 문제 카테고리 생성
- mbti 관련 문제 생성
  (wiki 상세 기입)
